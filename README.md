# IB Tools

## Build History API
Windows
```
Usage: build-history-api.exe [OPTIONS] --api-key <API_KEY> --coord-host <COORD_HOST> --coord-id <COORD_ID>

Options:
  -k, --api-key <API_KEY>
  -c, --coord-host <COORD_HOST>                        <hostname or ip>:8000
  -i, --coord-id <COORD_ID>
      --build-start-time-from <BUILD_START_TIME_FROM>
      --build-end-time-to <BUILD_END_TIME_TO>
      --initiator-id <INITIATOR_ID>
      --build-group <BUILD_GROUP>
      --build-title <BUILD_TITLE>
      --build-status <BUILD_STATUS>
      --build-duration-from <BUILD_DURATION_FROM>
      --build-duration-to <BUILD_DURATION_TO>
      --build-type <BUILD_TYPE>
  -h, --help                                           Print help
```
